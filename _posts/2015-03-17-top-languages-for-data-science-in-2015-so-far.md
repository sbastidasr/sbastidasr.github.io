**Market Share for each growing language:**

* R with 52.5% => 16% increase, from 45.1%.
* Python with 36.5% => 49% increase, from 24.6%
* Unix shell/awk/sed 14.5% => 44% increase, from 10.4%
* Lisp/Clojure 4.4% => 525% increase, from 0.7%

**Declining Languages:**

* Java (down 12%) 
* MATLAB (down 10%).
